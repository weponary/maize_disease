# Steps To Run This Project

**Clone**

```sh
  git clone git@github.com:udira-re/funolympic_frontend.git
```

## Run Locally

**Requirement**

- Node = v18.14.2

- Yarn = v1.22.19

**Steps**

- Open Project folder and run following command

```sh
    yarn
```

- Once completed run following command to start

```sh
  yarn run dev
```
